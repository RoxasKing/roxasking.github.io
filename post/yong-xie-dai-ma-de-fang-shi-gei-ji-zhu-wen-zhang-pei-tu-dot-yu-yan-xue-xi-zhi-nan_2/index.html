>>> 常用属性声明

所有属性声明皆为 键值对，即 name=value

strict digraph {
	Hello [color=green] // 为节点声明
	world [color=orange]
	Hello -> world [color=red] // 为边声明
	world -> Hello [arrowhead=diamond]
}
